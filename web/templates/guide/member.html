Getting Started

Welcome to the IPN Southeast Events Portal. Here's everything you need to know to get up and running.

{{/* Step 1 */}}
1

Sign In

Use the link from your welcome email to set your password, then sign in.

→ Go to the Set Password & Sign In link in your email

→ Choose a password (minimum 8 characters)

→ You'll be signed in automatically after setting your password

→ Next time: visit the portal and use Login with Email

{{/* Step 2 */}}
2

Submit an Event

Propose a new event for admin review.

→ Click + New Event in the top navigation

→ Fill out the 6-section form:

  • Basic Info — name, quarter, year, description, recurrence
  • Input — resources and preparation needed
  • Activities — what will happen at the event
  • Output — expected direct outputs
  • Outcome & Impact — longer-term effects
  • Support Requests — any help you need from the team

→ Click Submit for Review when ready

{{/* Step 3 */}}
3

Track Your Events

See the status of everything you've submitted.

→ Click My Events in the navigation to see all your submissions

→ Each event shows its current status:

Pending Approved Rejected

→ If rejected, you'll see the admin's feedback on the event page

{{/* Step 4 */}}
4

Edit an Event

Update a pending event before it's reviewed.

→ Open the event from My Events

→ Click Edit (only available while status is Pending)

→ Save your changes — the event will remain in review queue

Go to the Portal →
