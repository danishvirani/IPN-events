Getting Started

Welcome to the IPN Southeast Events Portal. As an Admin, you have full access to review events, manage the team, and plan the calendar.

{{/* Step 1 */}}
1

Sign In

Use Google SSO or the email/password you were set up with.

→ Visit the portal and click Continue with Google, or use Login with Email

→ You'll land on the Admin Dashboard showing a summary of all events

{{/* Step 2 */}}
2

Review Events

Approve or reject event submissions from team members.

→ Click All Events in the navigation to see all submissions

→ Filter by status: Pending, Approved, Rejected

→ Open any event and scroll to the Admin Review Panel

→ Add an optional comment, then click Approve or Reject

{{/* Step 3 */}}
3

Set Event Dates

Pin a confirmed date to any approved event.

→ Open an approved event's detail page

→ Find the Event Date panel below the review section

→ Pick a date and click Save Date

→ The date will appear in the Calendar and Roadmap views

{{/* Step 4 */}}
4

Manage Team Members

Control who has access and what they can do.

→ Go to Manage → Team Members in the navigation

→ New users appear automatically after their first Google sign-in

→ Change a user's role using the inline dropdown (Team Member / Viewer / Admin)

→ For password-based users, you can Set password directly or Send reset link to their email

Google SSO — signed in via Google

Password — email/password user

{{/* Step 5 */}}
5

Import Events via CSV

Bulk-create approved events from a spreadsheet.

→ Go to Tools → Import CSV in the navigation

→ Download the template and fill it in (17 columns, 3 required)

→ Upload the completed CSV — events are created as Approved immediately

→ Required columns: name, year, description

{{/* Step 6 */}}
6

Calendar & Roadmap

Plan and visualize the year ahead.

→ Planning → Calendar — month-by-month grid with event bars for recurring events

→ Planning → Roadmap — quarterly view of all approved events with dates

→ Use Download PDF on the Roadmap page to export a shareable report
